Objetivos especificos

Estos son los objetivos especificos a desarrollar durante la asignatura:

  • Crear un objeto de aprendizaje orientado a la formación de usuarios sobre el productor de ABP desarrollado, utilizando los sistemas de creación de contenidos visto en clase de eLearning.
  • Definir los objetivos que debe cumplir el asistente virtual.
  • Definir e implementar los intents que formarán parte de una conversación de dominio general.
  • Definir e implementar los intents relativos a la formación de usuarios e información sobre el producto de ABP.
  • Diseñar e implementar los flujos de conversación que implementen funcionalidades de la ABP y requieran conexión con servicios externos (API de la ABP y/o otros servicios de terceros).
  • Definir webhook de conexión con el API sobre https para que el asistente pueda resolver las intenciones del usuario del ABP. 
  • Implementar las funcionalidades del API necesarias para el asistente, estas deben estar recogidas en el documento de especificación de la ABP.
  • Integrar el asistente al menos en una plataforma de mensajería. Se puede integrar en las cuentas creadas para la difusión del grupo y/o del producto ABP.
  • Realizar las pruebas y entrenamiento necesario para mejorar las respuestas.
